# 🏥 MediCompare AI

## AI-Powered Hospital Comparison & Healthcare Decision Platform

MediCompare AI is a full-stack MERN healthcare platform that helps patients discover, compare, and select the best hospitals based on ratings, services, reviews, pricing, and location. The platform leverages Artificial Intelligence to provide intelligent hospital recommendations and review analysis, enabling users to make informed healthcare decisions.
