# MediCompare AI

AI-powered healthcare service comparison and appointment booking platform.

## Features

- User Authentication
- Hospital Management
- Service Management
- Service Search
- Hospital Comparison
- Appointment Booking (Coming Soon)
- Nearby Hospitals (Coming Soon)
- AI Recommendations (Coming Soon)

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Current Progress

- [x] Authentication
- [x] Hospital CRUD
- [x] Service CRUD
- [x] Service Search
- [x] Service Comparison
- [ ] Appointment Booking
- [ ] Maps Integration
- [ ] AI Recommendations

## Run Locally

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```
